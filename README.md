## 📋 Descripción
Una aplicacion web que imita una tienda en linea, cuenta con las siquientes caracteristicas:
1. Catalogo de Productos: Muestra productos con sus precios.
2. Carrito de Compras: Puedes agregar articulos y ver el costo total.
3. Formulario de Contacto: Para mandar mensajes, conectado a Firebase Firestore.
4. Autenticacion: Registro de usuarios con Firebase Auth.
5. Almacenamiento: Posibilidad de subir archivos usando Firebase Storage.

## 🛠️ Tecnologias Utilizadas
- React. js
- Bootstrap (Estilos CSS)
- Firebase (Firestore, Auth, Storage)
- Cordova (Para crear el archivo APK de Android)
